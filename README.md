# Trabalho-Gerenciamento

PASSO A PASSO - COMO EXECUTAR O PROJETO

1. CLONAR O REPOSITÓRIO

  git clone https://github.com/EduLippe/Trabalho-Gerenciamento.git

2. ACESSAR A PASTA DO PROJETO

  cd Trabalho-Gerenciamento

3. VERIFICAR A BRANCH (IMPORTANTE)

  git branch
  
  git checkout main - Se não estiver na main
  
4. INSTALAR DEPENDÊNCIAS 

 npm install

5. EXECUTAR O PROJET
        npm run dev

OBSERVAÇÕES:
- Certifique-se de ter Git instalado
- Certifique-se de ter Node.js
- Verifique se está na branch main
